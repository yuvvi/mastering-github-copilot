# 🧪 Unit Testing With AI

Instead of vague prompts like:

```
Write unit tests for this file
```

Use **focused prompts**.

Example:

```
Highlight a specific function → generate tests
```

Benefits:

* More accurate tests
* Higher coverage
* Cleaner test cases

Selecting **smaller code segments produces better AI output**.

# ⚙️ Advanced Copilot Capabilities

Copilot integrates deeply with developer workflows.

### Workspace AI

```
@workspace review my uncommitted changes
@workspace generate sequence diagram
```

### Terminal AI

```
@terminal profile app startup time
@terminal run only failed tests
```

### GitHub AI

```
@github summarize this PR
@github identify risks
@github check coding standards
```

These features transform Copilot into a **full development assistant** rather than just a code generator.
